Using Google Fonts

Remember that, for each Google font you want to use, you must include 2 lines of code in the head section of your page. The first line is a link element that includes the font information from Google's servers. The second is a font-family rule that tells the browser where you want to apply the font on the page. So, to include the font "Alegreya" and use it as the default font for all elements we would need to add this link line before our style section:

<link href='http://fonts.googleapis.com/css?family=Alegreya' rel='stylesheet' type='text/css'>

And then add the following rule to our style section:

body {
font-family: 'Alegreya', serif;
}

We could also add this line to a seperate style sheet, so long as we linked this seperate stylesheet After linking the Google Stylesheet with the font information. To add another font, say "Bitter" we'd need to add another stylesheet link, and then use the font-family rule to describe where this font should appear!

Border Radius

The border-radius property allows you to display boxes with rounded corners. The larger the value you set border-radius to, the larger the circle used to round off the corner, and the more oval-like, and less box-like, your box will become! Set a small value, relative to box size, for a square look. Set a large value, relative to box size, for a more oval look. For example:
